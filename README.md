# Gerador-de-Senhas

Esse é um gerador de senhas aleatórias, usando JavaScript ele é capaz de mustirar caracteres entre "a-z", "A-Z", "0-9", "!" e "?" <br>
podendo clicar na senha ele copia automáticamente para o clipboard do usuário.<br>
